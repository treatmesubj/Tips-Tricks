Pi Setup
	1. may be handy to have pi auto-connect to wifi
		a. will need new file in boot directory (/): "wpa_supplicant.conf"
			```
			ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
			update_config=1
			country=US
			network = {
			ssid="your wifi network name"
			psk="wifi password"
			key_mgmt=WPA-PSK
			}
			```
	2. will need an empty file in boot directory (/): "ssh"
	3. If Pi is booted, can enable ssh in raspi-config
		`sudo raspi-config`
			> 3. Interface Options
				> P2 SSH
					> Yes
					> Ok
			> Finish
	# default username:password = pi:raspberry
	# w/ sudo, can change password: `sudo passwd pi`

Finally, ssh in
	`ssh pi@raspberrypi.local`
